
## Project Description

We used Claude and this prompt:

> "you are a web developer. you are tasked with creating a simple series of web pages that will be used in an escape room. the index page is to replicate a web mail box.
> The mail box is completely non functional but must look like the real thing. only 12 emails in the displayed inbox can be clicked and opened. each email will need to look like a real email. the only functional items on the page are a report button and back to inbox.
> On clicking the report button, a popup screen will appear with a clue for the next part of the escape room. users must report suspicious emails.
> The product is for professionals and runs offline on one laptop."

We used Git Bash to commit and push our work to GitHub. The project includes multiple HTML, CSS, and JavaScript files, along with a provided logo.

---

## How to Run

1. Clone the repository
2. Open the folder
3. Open `index.html` in a browser

---

## Files

* `index.html` – inbox
* other `.html` files – emails
* `.css` – styling
* `.js` – functionality
* 'assets' - logo 

## Overview

in the browser, use local host to run the index.html file. all the emails and assigned letters in email.js can be changed.
